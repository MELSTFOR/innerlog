## CÓMO DESCARGAR COMO PDF

### Opción 1: Desde Chrome/Edge/Firefox
1. Abre cualquiera de los archivos HTML en el navegador
2. Presiona `Ctrl + P` (Windows) o `Cmd + P` (Mac)
3. Selecciona "Guardar como PDF"
4. Elige la carpeta y el nombre
5. Listo

### Opción 2: Desde el navegador (menú)
1. Abre el archivo en el navegador
2. Click derecho → "Imprimir" (o menú ≡ → "Imprimir")
3. Destino: "Guardar como PDF"
4. Ajusta márgenes si necesitas
5. Listo
