# Cómo Descargar INNERLOG como PDF

## Archivo Principal
**`resumen-de-app.html`** ← Abre este archivo

## Paso a Paso

### 1. Abre el archivo en tu navegador
- Haz doble clic en `resumen-de-app.html`
- O arrastra el archivo a tu navegador (Chrome, Edge, Firefox)

### 2. Descarga como PDF
Usa uno de estos métodos:

#### Método 1: Atajo de teclado (Recomendado)
```
Windows:  Ctrl + P
Mac:      Cmd + P
```

#### Método 2: Menú del navegador
1. Haz clic en el menú (≡ o tres líneas)
2. Selecciona "Imprimir"
3. En la ventana de impresión, selecciona "Guardar como PDF"

### 3. Configura el PDF
En la ventana de impresión:
- **Destino:** Guardar como PDF
- **Márgenes:** Mínimos (para mejor espacio)
- **Gráficos de fondo:** Activado (para que se vean bien las imágenes)
- **Orientación:** Vertical

### 4. Guarda el archivo
- Presiona "Guardar"
- Elige dónde guardar
- Dale un nombre (ej: "INNERLOG-Guia.pdf")
- ¡Listo!

## Características del PDF

✓ **Imágenes integradas** - Todas las capturas están incluidas
✓ **Profesional** - Diseño limpio y legible
✓ **Minimalista** - Sin emojis ni clichés
✓ **Completo** - 10 secciones + FAQ
✓ **Fácil de compartir** - Un solo archivo PDF

## Solución de Problemas

### Las imágenes no aparecen en el PDF
**Solución:** En la ventana de impresión, busca la opción "Gráficos de fondo" o "Imágenes de fondo" y actívala.

### El PDF se ve cortado
**Solución:** 
- Reduce los márgenes en la ventana de impresión
- Usa "Escala": 100%
- Intenta con "Papel personalizado" si es necesario

### El tamaño del archivo es muy grande
**Solución:** Algunos navegadores comprimen automáticamente. Si quieres reducirlo más:
- Abre el PDF en Adobe Reader
- Archivo → Optimizar PDF
- Reduce calidad de imagen si es necesario
