# Instrucciones para las Imágenes

El archivo `resumen-de-app.html` necesita que las imágenes estén organizadas en una carpeta llamada `images`.

## Estructura de carpetas

```
server/
├── resumen-de-app.html
└── images/
    ├── 01-login.png
    ├── 02-wellness.png
    ├── 03-sesion.png
    ├── 04-historial.png
    ├── 05-tests.png
    ├── 06-comunidad.png
    ├── 07-dashboard-atleta.png
    ├── 08-psicologo.png
    └── 09-psicologo-atletas.png
```

## Cómo preparar las imágenes

1. **Crea la carpeta** `images` en la misma carpeta donde está `resumen-de-app.html`
2. **Guarda cada captura** con el nombre correspondiente:
   - `01-login.png` → Pantalla de Bienvenido (Login)
   - `02-wellness.png` → Check-in Wellness
   - `03-sesion.png` → Registro de Sesión
   - `04-historial.png` → Historial
   - `05-tests.png` → Tests Cognitivos
   - `06-comunidad.png` → Comunidad
   - `07-dashboard-atleta.png` → Dashboard del Atleta (Hola, Melina)
   - `08-psicologo.png` → Dashboard Psicólogo
   - `09-psicologo-atletas.png` → Vista de Psicólogo Deportivo

## Cómo visualizar

1. Abre `resumen-de-app.html` en tu navegador
2. Las imágenes deberían aparecer en cada sección
3. Para descargar como PDF:
   - Presiona `Ctrl + P` (Windows) o `Cmd + P` (Mac)
   - Selecciona "Guardar como PDF"

## Características del documento

✓ **Minimalista** - Sin emojis, sin clichés
✓ **Profesional** - Lenguaje claro y directo
✓ **Responsivo** - Se adapta a cualquier pantalla
✓ **Optimizado para PDF** - Saltos de página automáticos
✓ **Navegación** - Enlaces rápidos en el menú superior

---

Si alguna imagen no carga, verifica que:
- La carpeta `images` esté en la misma ubicación que el HTML
- Los nombres de archivo sean exactamente como se especificó (respetando mayúsculas y extensión)
- Las imágenes sean archivos PNG válidos
