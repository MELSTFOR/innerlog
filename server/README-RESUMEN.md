## Paso a Paso

### 1. Prepara las imágenes
Guarda las 9 capturas con estos nombres exactos en la carpeta `images/`:

```

02-wellness.png ............... Check-in Wellness
03-sesion.png ................. Registro de Sesión
04-historial.png .............. Historial

08-psicologo.png .............. Dashboard Psicólogo

```

### 2. Abre el archivo
- Haz doble clic en `resumen-de-app.html`
- O arrastra el archivo al navegador

### 3. Descarga como PDF
- Presiona `Ctrl + P` (o `Cmd + P` en Mac)
- Selecciona "Guardar como PDF"
- Elige la ubicación
- Listo

## Solución de Problemas

### Las imágenes no aparecen
- Verifica que la carpeta `images` esté en la misma ubicación que el HTML
- Revisa que los nombres sean exactos (mayúsculas, guiones, extensión .png)
- Abre la consola (F12) para ver mensajes de error

### El PDF se ve mal
- Asegúrate de usar Chrome, Edge o Firefox
- En configuración de impresión, activa "Gráficos de fondo"
- Reduce márgenes si necesitas más espacio

### Falta alguna sección
- Abre el archivo en el navegador
- Usa el menú de navegación superior
- Los enlaces van directo a cada sección
